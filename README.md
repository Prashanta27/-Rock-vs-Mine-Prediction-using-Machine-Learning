# -Rock-vs-Mine-Prediction-using-Machine-Learning
🚢 Rock vs Mine Prediction using Machine Learning  This project builds a Machine Learning classification model that predicts whether an underwater object is a Rock or a Mine using Sonar signal data.
Submarines and underwater vehicles use sonar technology to detect objects beneath the ocean surface. The sonar system sends sound waves and measures the reflected signals from objects. These reflected signals are converted into 60 numerical features, which represent the energy of sonar waves at different frequency bands.

Using these 60 sonar features, the trained Machine Learning model learns patterns that differentiate natural rocks from dangerous underwater mines.
🔬 Dataset

The dataset contains sonar signals reflected from different underwater objects.
Features: 60 sonar frequency energy values
Target:
R → Rock
M → Mine
Total Samples: 208
